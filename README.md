# pruning-rml

Revision demystifies pruning: notebooks and lab reports exploring reset, revision, and residues in small language models.

This starter bundle includes:

- `paper.yaml`
- `notebooks/00_context.ipynb`
- `src/pruning_rml/`
- `requirements.txt`

Start with:

```bash
pip install -r requirements.txt
jupyter notebook notebooks/00_context.ipynb
```
